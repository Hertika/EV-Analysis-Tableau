# Electric Vehicle Analysis Dashboard 

## Project Description
This project focuses on analyzing Electric Vehicle (EV) adoption trends using Tableau. The dashboard provides an interactive and visual representation of EV data, helping users understand key patterns such as vehicle type distribution, top manufacturers, and growth over time.

The dataset was processed and visualized to extract meaningful insights that can support data-driven decision-making in the EV industry.

##  Key Features
- Interactive dashboard with dynamic filters
- Comparison of Battery Electric Vehicles (BEV) and Plug-in Hybrid Electric Vehicles (PHEV)
- Analysis of top EV manufacturers like Tesla, Nissan, and Chevrolet
- Year-wise growth trends of EV adoption
- Geographic visualization of EV distribution using maps

## Tools & Technologies
- Tableau (Data Visualization)
- Microsoft Excel (Data Source)

## Insights Gained
- BEVs dominate the EV market compared to PHEVs
- Tesla is the leading EV manufacturer
- Significant growth in EV adoption after 2018
- Certain regions show higher EV concentration
